## Shiawase

Verge's frontend, basically ripple but way better and faster.

## Installation
Copy config.sample.php as config.php and edit it
```
$ cd inc
$ cp config.sample.php config.php
$ nano config.php
```
Then, run composer install on the main directory
```
$ composer install
```
